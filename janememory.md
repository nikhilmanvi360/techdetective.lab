## 🎭 1. Visual & Aesthetic Identity
The platform has been unified under a single **Pixel-Perfect Detective Noir** design system.

*   **Primary Palette**:
    *   **Dark Walnut**: `#140e06` (Backgrounds, Depth)
    *   **Bronze**: `#a07830` (Borders, Accents, HUD Chrome)
    *   **Aged Parchment**: `#f0e0a0` (Panels, Content surfaces)
    *   **Ink**: `#1a0e04` (Typography, High-fidelity contrast)
*   **Aesthetic Rules**:
    *   **Synchronized HUD**: Every core page features the exact same Bronze Top-Chrome bar (Logo, XP, Rank, Disconnect).
    *   **Cinematic Motion**: All transitions utilize Framer Motion with ease-out cubic springs for a heavy, tactile feel.
    *   **Game-Life Icons**: Lucide icons are used exclusively in monochromatic bronze or high-contrast ink.

## ⚙️ 2. Core Architecture
A scalable, real-time forensic engine designed for beginner hacking competitions.

*   **Frontend**: React (Vite) + Tailwind CSS (Core Framework) + Motion. 
*   **Backend**: Node.js/Express + Socket.IO (Dynamic event streaming).
*   **Database**: Supabase (PostgreSQL) tracking all durables (Teams, Scores, Puzzles, Evidence, Badges).
*   **Handshake/AI**: @heyputer/puter.js manages neural links and strategic AI consulting.
